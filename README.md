# 💳 Credit Risk Prediction System

## Overview
This project is a Machine Learning-based Credit Risk Prediction system that predicts whether a loan applicant is likely to be a good or bad credit risk based on financial and personal data.

The model is deployed using Streamlit for real-time user interaction.

---

## Machine Learning Model
- Algorithm: Random Forest Classifier (or your model)
- Dataset: German Credit Dataset
- Approach: Classification problem (Good / Bad credit risk)

---

## Features Used
- Age
- Credit Amount
- Loan Duration
- Payment to Income Ratio
- Job Type
- Housing Status
- Account Status
- Number of Credits
- Residence Duration

---

## How to Run the project

Install dependencies:
pip install -r requirements.txt

Run app:
streamlit run app.py

## Project Info
Machine learning model predicts credit risk using financial data.
